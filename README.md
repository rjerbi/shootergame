# Space Shooter

**Space Shooter** is a 2D space shooter game developed in Python using Pygame. The player controls a spaceship to destroy enemies while avoiding collisions. The game includes randomly generated enemies, laser shooting, explosions, a scoring system, and player lives.

---

## Features

- Control the spaceship with left/right arrow keys.
- Shoot lasers using the Enter key.
- Randomly generated enemies with increasing difficulty.
- Health points and lives system.
- Collision detection: bullet-enemy and spaceship-enemy.
- Background music and sound effects.
- Animated explosions when enemies are destroyed.
- Game Over screen with final score and retry option.

---

## Requirements

- Python  
- Pygame
